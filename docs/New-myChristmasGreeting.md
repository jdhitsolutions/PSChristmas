# New-myChristmasGreeting

## SYNOPSIS
Create a Christmas greeting.

## SYNTAX

```
New-myChristmasGreeting
```

## DESCRIPTION
This command will return a random Christmas greeting, most likely "Merry Christmas" from another language.

## EXAMPLES

### Example 1
```
PS C:\> New-MyChristmasGreeting
Meri Kuri
```
