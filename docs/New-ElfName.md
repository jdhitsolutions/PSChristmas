# New-ElfName

## SYNOPSIS
Find your inner elf

## SYNTAX

```
New-ElfName
```

## DESCRIPTION
This command will provide you an elf name.

## EXAMPLES

### Example 1
```
PS C:\> New-Elfname
Elsa Mintypuff
```
